Проект находится в стадии рефакторинга.
Документация и комментарии будут позднее.

Краткое описание.
Простейший файловый менеджер.
Возможности: 
показ содержимого дисков;
создание папок/файлов;
удаление папок/файлов;
переименовывание папок/файлов;
копирование/перенос папок/файлов;
вычисление размера папки/файла;
производить поиск (с поиском по подпапкам);
